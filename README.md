# OpenGL Triangle

This project is a simple OpenGL application that renders a triangle using the modern graphics pipeline. It serves as my starting point for learning computer graphics and understanding how rendering works at a low level.

## What I Learned

- Setting up an OpenGL context using GLFW and GLAD  
- Understanding the rendering pipeline (vertex processing → fragment processing)  
- Creating and using Vertex Buffer Objects (VBO)  
- Working with Vertex Array Objects (VAO)  
- Writing and compiling basic shaders (vertex and fragment shaders)  
- Passing data from CPU to GPU  
- Rendering primitives using `glDrawArrays`  

## Technologies Used

- C++  
- OpenGL  
- GLFW  
- GLAD  

## Purpose

The goal of this project was to build a strong foundation in OpenGL by starting with the most basic rendering task of drawing a triangle and understanding each step involved in the process.
